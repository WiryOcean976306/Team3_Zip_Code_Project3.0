AI Use Statement
================

Course: Introduction to Operating Systems
Project: Team 3 Zip Code Project 3.0
Date: 2026-04-06

Summary
-------
This project used AI assistance as a development support tool. AI was used to help with code refactoring, debugging, command-line interface cleanup, and documentation drafting.

How AI Was Used
---------------
1. Reviewed and explained existing C++ code structure and behavior.
2. Suggested and implemented targeted code edits
3. Helped with project documentation

How AI Was NOT Used
-------------------
1. AI did not independently submit work.
2. AI did not replace team decision-making.
3. AI outputs were not accepted blindly; code and results were reviewed and tested in the local environment.

Verification and Responsibility
-------------------------------
All outputs, and submitted materials were reviewed by the team. The team is responsible for correctness, testing, academic integrity, and final submission content.

Acknowledgment
--------------
AI assistance was used similarly to an advanced coding tutor/tool to improve productivity and clarity while preserving student ownership of the project.
